# tech_talk_demo

Demo project for showcasing an agent-driven refactor workflow ("Ralph" loop) on a Java codebase.

## Repository Layout

- `src/`: Application and test code.
- `ralph_demo/`: Talk artifacts and loop materials (`ralph.sh`, `prompt.md`, `prd.json`, `progress.txt`).

## Quick Start

```bash
./gradlew test
```

For the full demo walkthrough, see `ralph_demo/README.md`.
